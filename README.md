# iPSX3

**iPSX3** is an experimental iOS port of **RPCS3**, focused on bringing PlayStation 3 emulation research to iPhone.

> Release date: **TBD**

## ✨ Overview

| Item         | Status                       |
| ------------ | ---------------------------- |
| Project name | **iPSX3**                    |
| Platform     | iPhone / iOS                 |
| Base project | RPCS3                        |
| Renderer     | Metal backend in development |
| Release      | Not announced yet            |

## 🚧 Current Status

iPSX3 is under active development and is not ready for public release yet.

Current work includes:

* Real-game boot and gameplay testing
* Metal backend implementation
* iPhone device testing
* iOS-specific memory and performance optimization
* Compatibility and stability improvements
